Real-Time Healthcare Monitoring System

Application 6 Project by Sara Lucia Vargas-Cortes

Project Overview

This project is a real-time healthcare monitoring system implemented using FreeRTOS on an ESP32. It simulates a heart sensor and detects abnormal values in real time, triggering an alarm system.

Real-Time Task Structure

Sensor Task

Reads sensor every 100 ms.
