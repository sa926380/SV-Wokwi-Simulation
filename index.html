Application 6 Real-Time Hardware Demo

Physical embedded system prototype using real-time task scheduling

Project Overview

This project is a real-time embedded system implemented on physical hardware. The system monitors inputs, responds to events, and demonstrates predictable timing using task scheduling.

Demo Video

The video below shows the physical hardware running, including the board, sensor input, LED response, and serial timing logs.

Key Features

  • Physical hardware implementation
  • Real-time task scheduling
  • Sensor-triggered system response
  • LED indicators for system status
  • Timing verification using serial logs

What I Learned

This project helped me understand how real-time systems organize different tasks so important events can be handled on time. I learned how hardware inputs, software timing, and task priorities work together in an embedded system.
