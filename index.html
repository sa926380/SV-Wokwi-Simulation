Real-Time Healthcare Monitoring System

Application 6 Project | Sara Lucia Vargas-Cortes

Project Overview

This project is a real-time healthcare monitoring system implemented using FreeRTOS on an ESP32. The system reads a simulated heart sensor using a potentiometer and detects when the signal exceeds a critical threshold.

System Architecture

Sensor Task

Runs every 100 ms to detect abnormal values.

Hard Real-Time High Priority
Event Handler Task

Handles alerts and controls alarm behavior.

Hard Real-Time High Priority
Button Task (ISR)

Detects user acknowledgement immediately using interrupts.

Hard Real-Time
Heartbeat Task

Blinks a green LED to indicate system is alive.

Soft Real-Time Low Priority

Key Features

Real-Time Scheduling

Task priorities ensure critical tasks always meet deadlines.

Synchronization

Mutex prevents race conditions in serial output.

Event Detection

Counting semaphore tracks sensor-triggered events.

Interrupt Handling

Binary semaphore + ISR ensures immediate button response.

Safe Data Transfer

Queue used to pass sensor data between tasks.

Deterministic Timing

Response time is within a few milliseconds (well below 150 ms deadline).

Persistent Alarm

Red LED continues blinking until user acknowledges the alert.
