Real-Time Healthcare Monitoring System

Application 6 Project by Sara Lucia Vargas-Cortes

Project Overview

This project is a real-time healthcare monitoring system implemented using FreeRTOS on an ESP32. The system reads a simulated heart sensor using a potentiometer and detects abnormal values in real time, triggering an alarm system.

Real-Time Task Structure

Sensor Task

Runs every 100 ms to detect abnormal values.

Hard RT Priority 2
Event Handler Task

Handles alerts and controls the alarm.

Hard RT Priority 2
Button Task

ISR-triggered acknowledgement system.

Hard RT Priority 3
Heartbeat Task

Shows system is alive using LED.

Soft RT Priority 1

Key Features

Deadline Guarantee

High-priority tasks meet strict timing requirements.

Mutex Protection

Prevents race conditions in serial printing.

Counting Semaphore

Tracks multiple sensor alert events.

Binary Semaphore + ISR

Immediate response to button press.

Queue Communication

Transfers sensor data safely between tasks.

Persistent Alarm

Alarm continues until user acknowledges it.

Wokwi Hardware Simulation

This embedded simulation shows the ESP32 system with the potentiometer sensor, LEDs, and button used to test real-time behavior.
